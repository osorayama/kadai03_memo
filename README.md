# ①課題名
登山ログ 〜成層圏突破チャレンジ〜

## ②課題内容（どんな作品か）
- 日本全国の山を記録しながら、累積標高で「成層圏→中間圏→熱圏」突破を目指すWebアプリです。
- 山を追加・管理し、合計標高を自動集計。
- 都道府県ごとの達成状況を日本地図で可視化。

## ③アプリのデプロイURL
https://osorayama.github.io/kadai03_memo/

## ④アプリのログイン用IDまたはPassword（ある場合）
- ログイン不要

## ⑤工夫した点・こだわった点
- 累積標高による達成感の演出（成層圏・中間圏・熱圏の段階）
- 日本地図で都道府県ごとの進捗を可視化
